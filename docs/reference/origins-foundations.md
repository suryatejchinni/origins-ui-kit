# Origins Foundations Components

Components that visualize the design tokens and foundations of the system. These are primarily useful for documentation sites or style guides.

## ColorPalette

Displays the current color palette based on the active theme.

### Import

```tsx
import { ColorPalette } from "origins-ui-kit"
```

### Usage

```tsx
<ColorPalette />
```

---

## SpacingRadii
